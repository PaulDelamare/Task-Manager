# Gestionnaire de tâches

Application React pour gérer vos tâches quotidiennes.

## Fonctionnalités

- Connexion
- Création de compte
- Créer une tâche
- Modifier le contenu d’une tâche
- Marquer une tâche comme terminée
- Supprimer une tâche

## Prérequis

- Node.js 18+
- npm (installé avec Node.js)

## Démarrage rapide

```bash
npm install
npm run dev
```

Ouvrez ensuite l’URL indiquée dans le terminal (en général http://localhost:5173).
